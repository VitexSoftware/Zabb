### Full Description
Zabb is a Flutter-based mobile client for Zabbix monitoring system, designed to bring enterprise IT infrastructure monitoring to your mobile device.

**Key Features:**
- Real-time problem monitoring with auto-refresh
- Advanced filtering by severity and hostname
- Problem acknowledgment and closure
- Trigger disable functionality
- Pull-to-refresh for instant updates
- Audio notifications with customizable sounds per severity
- Recovery sound notifications
- Multi-platform support (Android, iOS, Desktop, Web)
- SSL support for self-signed certificates

**Platform Features (Mobile & Desktop):**
- Background monitoring
- System notifications
- Custom sound file uploads
- Battery optimization management

Perfect for system administrators and IT professionals who need to monitor their Zabbix infrastructure on the go.
